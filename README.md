## Visualizing The Markov Chain: Chord Progressions
![MARKOVCHAIN](https://github.com/user-attachments/assets/22614c52-aa41-431e-93b1-f87039a5690d)
Using stochastic statistical methods, I compare two of my favorite groups by their chord progression usage.
Transition probability matrices for each artist showcase patterns in melodies/chords.

## Survival Analysis: Applying the Cox Regression & Other Survival Methods on Hospitalization Data
Using SAS, this project applies the cox regression model, stratified cox model, and other statistical methods such as the Log-Rank Test and Kaplan Meier curves to help visualize and propose a final model for a heart attack study.

Below is a table of the model building process with variables (with * indicating significance) and the corresponding 2logL statistic, and AIC.
